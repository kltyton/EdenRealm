# 伊甸之境 / Eden Realm 设计数据库

本文件是后续内容实现依据。当前框架阶段只实现少量可验证示例内容，不把完整数据库硬编码进注册类。

## 当前已落地

- 方块：22 套树木基础方块，包含原木、木、去皮原木、去皮木、木板、楼梯、台阶、栅栏、栅栏门、按钮、压力板、木架、树叶、树苗、门、活板门、告示牌、墙上告示牌、悬挂告示牌、墙上悬挂告示牌。
- 物品：上述可持有方块物品，以及 22 套普通船、运输船。
- 实体：22 个普通船实体、22 个运输船实体，复用原版船行为。
- 创造模式标签页：`eden_realm`
- 数据生成：语言、blockstate/model、item model/client item、recipe、loot、block/item/entity tags
- 地形与生态方块：44 个地形方块、7 个植物方块、6 套共 48 个珊瑚方块。
- 伊甸草方块：只注册 `eden_grass_block` 一个方块；同一方块按权重随机选择基础顶面、两种替换顶面和三种顶部装饰外观，并随机应用四个水平旋转。

伊甸草方块使用 `eden_realm:textures/colormap/grass.png` 的独立 256x256 色图。顶部草底色和侧面草覆盖层参与群系着色，三张顶部装饰层保持原色，不参与 tint。伊甸泥土、伊甸草方块、伊甸草径和伊甸耕地已形成铲平、锄耕、退化和湿度更新链路。

当前珊瑚集合：蜂孔、管状、球状、海花、枝状、扇形。每种包含活体/失活的珊瑚块、珊瑚植株、珊瑚扇和墙上珊瑚扇；墙上变体与原版一致，不单独注册物品，而由对应珊瑚扇物品根据放置面选择落地或墙上方块。

当前植物行为：浮萍采用原版睡莲的水面支撑与专用放置物品；寒晶草、霜绒花、耐旱短草、刺枝灌木和沙原短草使用按贴图可见范围定义的选择轮廓；腐木菌毯保留透明区域且不遮挡下方方块顶面。
